# ChzEx

Elixir port of OpenAI's [chz](https://github.com/openai/chz) library - a powerful configuration management system for building composable, type-safe command-line interfaces with hierarchical configuration, environment variable support, and flexible argument parsing.

## Features

- Composable configuration definitions
- Type-safe configuration parsing
- Hierarchical configuration with inheritance
- Environment variable support
- Flexible command-line argument parsing

## Installation

Add `chz_ex` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:chz_ex, "~> 0.1.0"}
  ]
end
```
